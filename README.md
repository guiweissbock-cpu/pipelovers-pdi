# Como usar este repositório

## Atualizar o dashboard

1. Vá em **"Add file" → "Upload files"**
2. Suba os dois arquivos:
   - Planilha de PDIs → renomeie para **`pdi.csv`**
   - Relatório de consumo → renomeie para **`consumo.csv`**
3. Clique em **"Commit changes"**
4. Aguarde ~2 minutos
5. O dashboard estará atualizado em:

**https://SEU_USUARIO.github.io/pipelovers-pdi/**

---

Nenhum código necessário. O GitHub faz tudo automaticamente.
